<h3>Описание</h3>
<p>Программа выводит на экран фигурку, позволяет ею манипулировать: перемещение, поворот, масштабирование.
Управление фигуркой представлено ниже:
</p>
<ul>
<li>Перемещение при помощи стрелок: UP, RIGHT, DOWN, LEFT</li>
<li>Поворот: Q - влево, E - вправо</li>
<li>Масштабирование: 1 - уменьшение, 2 - увеличение</li>
</ul>
